Informed Consent

Purpose of the study

You are invited to take part in a research study on decision-making and cognitive tasks. The study is conducted by researchers at a university and is intended to understand how people perform symbol-matching tasks under timed conditions.

What you will do

{% if condition == 'no_break' %} You will complete a symbol-matching task in two work periods: a first session of {{ seg1_total_minutes }} minutes followed by a second session of {{ task_minutes }} minutes, with no break in between. {% elif condition == 'forced_break' %} You will complete a symbol-matching task in two sessions of {{ task_minutes }} minutes each, separated by a required {{ break_minutes }}-minute break. {% else %} You will complete a symbol-matching task in two sessions of {{ task_minutes }} minutes each, with an optional {{ break_minutes }}-minute break in between. {% endif %} On each trial you will see a target symbol and must select all matching symbols from a grid of 64.

Payment

You will receive a show-up fee of $3.00 for completing the study. In addition, you earn experiment tokens for every correct task:

  • {% if condition == 'no_break' %} Session 1 ({{ seg1_total_minutes }} minutes): {% elif condition == 'choice' %} Session 1 (and the break period if you work through it): {% else %} Session 1: {% endif %} 1 token per correct task — each token is worth $0.004.
  • {% if condition == 'no_break' %} Session 2 ({{ task_minutes }} minutes): {% else %} Session 2: {% endif %} 2 tokens per correct task — each token is worth $0.004, so $0.008 per correct task.

Your total earnings will be paid to you at the end of the session.

Risks and benefits

There are no known risks beyond those of everyday computer use. There is no direct personal benefit beyond the payment described above, but your responses will contribute to scientific research.

Confidentiality

All data are stored securely and will only be reported in anonymised, aggregated form. Your name will never be linked to your responses in any publication. Your participant ID (not your name) will be used to identify your data.

Voluntary participation

Participation is entirely voluntary. You may stop at any time without penalty, though you will only receive the full payment if you complete both sessions. If you have questions, please ask the research assistant before proceeding.
